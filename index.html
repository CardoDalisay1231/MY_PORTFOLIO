Enrollment System for CVsU - Bacoor

This collaborative, four-member group project delivers a custom web-based enrollment system for Cavite State University - Bacoor Campus. Built with React.js & Bootstrap on the frontend and Node.js (Express) with PostgreSQL on the backend, it automates the entire student registration workflow—from course selection and fee payment to document upload—with real-time validation and responsive design. Role-based access control keeps data secure, Axios handles all HTTP requests, and every API endpoint is documented using Swagger to streamline future maintenance.
